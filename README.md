# Create With × EVOLVE — Hackathon Kickoff Deck

A self-contained HTML presentation for the Create With × EVOLVE hackathon kickoff call.
Built in the Create With brand: zero dependencies, all CSS/JS inlined — it runs entirely in the browser.

## View it
- **Locally:** open `index.html` in any browser (no server needed).
- **Online:** if GitHub Pages is enabled for this repo, the deck is served at the Pages URL
  (the deck is `index.html` at the repo root).

## Navigate
- **← →** arrows, **Space**, scroll, or swipe between slides
- Click the dots on the right to jump to a slide
- **F** toggles fullscreen (handy when screen-sharing on Zoom)

## Contents
```
index.html                   # the presentation (self-contained)
hackathon-kickoff-assets/    # logos and photos referenced by the deck
```

> Fonts load from Google Fonts (needs internet — fine on a call). Images are referenced by
> relative path from `hackathon-kickoff-assets/` and work offline.
